# BankManagementSystem
This Bank Management System is a Python-based mini project designed to perform basic banking operations in a simple and user-friendly way. 
The system allows users to create accounts, log in securely using a username and password, deposit money, withdraw money, and view mini statements containing account details and current balance. 
It is developed using object-oriented programming concepts such as classes, objects, methods, and encapsulation for better data management and security. 
The project also includes menu-driven interaction and basic error handling to improve usability. 
This system helps beginners understand the practical implementation of banking functionalities in Python while improving their programming logic and problem-solving skills. 
It is an ideal project for students to learn core Python concepts and real-world application development.
